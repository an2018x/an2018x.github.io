P1149 火柴棒等式(搜索)

题目：

给你n根火柴棍，你可以拼出多少个形如”A+B=C”的等式？等式种的A,B,C是用火柴棍拼出的整数（若该数非零，则最高位不能是0）。用火柴棍拼数字0~9的拼法：

注意：

1、加号与等号各自需要两根火柴棍

2、如果$A\neq B$，则$A+B=C$与$B+A=C$视为不同的等式（$A,B,C \geq 0$）

3、n根火柴棍必须全部用上

输入：

一个整数n($n\leq 24$)。

输出：

能拼成的不同等式的数目。

输入：

14

输出：

2

解：

#include<bits/stdc++.h>
using namespace std;
int cnt[10]={6,2,5,5,4,5,6,3,7,6};
int n;
int main(){
    cin>>n;
    int j;
    int a[2001];
    int _cnt=0;
    a[0]=6;
    for(int i=1;i<=2000;i++){
        j=i;
        while(j>=1){
            a[i]=a[i]+cnt[j%10];
            j/=10;
        }
    }
    for(int i=0;i<=1000;i++)
        for(int j=0;j<=1000;j++)
            if(a[i]+a[j]+a[i+j]+4==n) _cnt++;
    cout<<_cnt<<endl;
    return 0;
}

            } else

            n = n + x[i];

P1036（选数）

题目：

已知n个整数$x_1,x_2,…x_n$，以及1个整数k(k<n)。从n个整数种任选k个整数相加，可分别得到一系列的和。现在要求你计算出和为素数共有多少种。

输入：

第一行：$n,k,(1\leq n \leq 20,k<n)$

第二行：$x_1,x_2,…x_n$$(1\leq x_i\leq 5000000)$

输出：

满足条件的整数。

输入：

4 3
3 7 12 19

输出：

1

解：

搜索种组合数的问题

#include <bits/stdc++.h>
using namespace std;
int f[21];
int n,k;
int vis[21];
int _cnt=0;
int sum=0;
int d[21];
bool is_prime(int num){
    int s=sqrt(double(num));
    for(int i=2;i<=s;i++){
        if(num%i==0) return false;
    }
    return true;
}
void dfs(int i,int st){
    if(i==k){
        if(is_prime(sum)) _cnt++;
        return;
    }
    for(int j=st;j<n;j++){
            d[i]=f[j];
            sum+=f[j];
            dfs(i+1,j+1);
            sum-=f[j];

    }
}
int main(){
    sum=0;
    memset(vis,0,sizeof(vis));
    cin>>n>>k;
    for(int i=0;i<n;i++){
        cin>>f[i];
    }
    _cnt=0;
    dfs(0,0);
    cout<<_cnt<<endl;
    return 0;
}

P1217（回文质数）

题目：

找出$a,b$内的所有回文质数。

输入：

俩个整数a,b。

输出：

回文质数表，一行一个。

输入：

5 500

输出：

5
7
11
101
131
151
181
191
313
353
373
383

解：

反转用$stl$里的reverse,质数判断用素数筛。

打表：

#include <bits/stdc++.h>
using namespace std;
int ans[2000]={所有数据};
int main(){
    int a,b;
    cin>>a>>b;
    for(int j=0;j<2000;j++){
        if(ans[j]>=a&&ans[j]<=b) cout<<ans[j]<<endl;
        if(ans[j]>b) break;
    }
    return 0;
}
