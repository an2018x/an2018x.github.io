P1031 均分纸牌 （模拟、贪心、概率论、统计）

题目：

有N堆纸牌，编号分别为1，2，…，N。每堆上有若干张，但纸牌总数必为N的倍数。可以在任一堆上取若干张纸牌，然后移动。

移牌规则为：在编号为1堆上取的纸牌，只能移到编号为2的堆上；在编号为N的堆上取的纸牌，只能一到编号为N-1的堆上；其他堆上取的纸牌，可以移到相邻左边或者右边的堆上。

现在要求找出一种移动方法，用最少的移动次数使每堆上纸牌数都一样多。

例如N=4，4堆纸牌数分别为：

9 8 17 6

移动3次可以达到目的：

A (3) 4->(4) B (3) 3->(2) C (2) 1->(1)

输入格式：

第一行：N(N堆纸牌，1<=N<=100)

第二行：$A_1,A_2,…,A_n$(N堆纸牌，每堆纸牌初始数，($1\leq A_i \leq 10000$)

输出格式：

一行：最少移动次数。

解 1：1、算平均数

2、求每堆纸牌与平均数的关系（多记为正，少记为负）

3、当$q[y]$（第y堆纸牌与平均数的关系）不等于0时，$q[y+1]=q[y+1]+q[y]$，移动次数加1

#include <bits/stdc++.h>
using namespace std;
int q[10010];
int main(){
    int N,cnt=0,ave=0;
    cin>>N;
    for(int i=0;i<N;i++){
        cin>>q[i];
        ave+=q[i];
    }
    ave/=N;
    for(int i=0;i<N;i++)
        q[i]-=ave;
    for(int i=0;i<N;i++){
        if(q[i]!=0){
            q[i+1]+=q[i];
            cnt++;
        }
    }
    cout<<cnt;
    return 0;
}

P1042 乒乓球（模拟，字符串）

题目：

小明想要弄明白乒乓球11分制和21分制对选手的影响，对多年比赛的统计数据进行分析。首先他将比赛每个球的胜负列成一张表，然后分别计算在11分制和21分制下，双方的比赛结果。

现在有一份数据，其中W表示小明得一分，L表示小明对手得一分。

WWWWWWWWWWWWWWWWWWWWWWLW

在11分制下，此时的结果是第一局小明11比0获胜，第二局11比0获胜，正在进行第三局，当前比分1比1。而在21分制下，此时比赛结果是小明第一局21比0获胜，正在进行第二局，比分2比1。如果一局比赛刚开始，则此时比分为0比0。直到分差大于或等于2，才一局结束。

对于一系列比赛信息的输入(WL形式)，输出正确的结果。

输入格式：

每个输入文件包含若干行字符串，字符串有大写的W、L和E组成。其中E表示比赛信息结束，程序应该忽略E之后的所有内容。

输出格式：

输出由两部分组成，每部分有若干行，每一行对应一局比赛的得分（按比赛信息输入顺序）。其中第一部分是11分制下的结果，第二部分是21分制下的结果，两部分由一个空行分隔。

输入：

WWWWWWWWWWWWWWWWWWWW
WWLWE

输出：

11:0
11:0
1:1

21:0
2:1

解：

1、E不一定出现在文件的末尾

2、比赛要领先2个球才能获胜

#include <bits/stdc++.h>
using namespace std;
string s[100001];
int a[100001], b[100001], a2[100001], b2[100001];
bool check(int x) {
    int t = s[x].size();
    for (int i = 0; i < t; i++)
        if (s[x][i] == 'E') return 0;
    return 1;
}
int main() {
    int n = 1, i, j, t = 1, t2 = 1;
    cin >> s[n];
    while (check(n)) {
        n++;
        cin >> s[n];
    }
    for (i = 1; i <= n; i++) {
        int sz = s[i].size();
        for (j = 0; j < sz; j++) {
            if (s[i][j] == 'E') {
                for (i = 1; i <= t; i++) cout << a[i] << ":" << b[i] << endl;
                cout << endl;
                for (i = 1; i <= t2; i++) cout << a2[i] << ":" << b2[i] << endl;
                return 0;
            }

            if (s[i][j] == 'W')
                a[t]++, a2[t2]++;
            else if (s[i][j] == 'L')
                b[t]++, b2[t2]++;
            if ((a[t] >= 11 || b[t] >= 11) && abs(a[t] - b[t]) >= 2) {
                t++;
            }
            if ((a2[t2] >= 21 || b2[t2] >= 21) && abs(a2[t2] - b2[t2]) >= 2) {
                t2++;
            }
        }
    }
    return 0;
}

解：

#include <algorithm>
#include <cstdio>
#include <cstring>
using namespace std;
int T;
int n, r;
char ch[100];
int main() {
    scanf("%d", &T);
    while (T--) {
        scanf("%d%d", &n, &r);
        int x = abs(n), y, cnt = 0;
        if (x == 0) {
            printf("0\n");
            continue;
        }
        while (x != 0) {
            y = x % r;
            x = x / r;
            if (y < 10)
                ch[cnt++] = y + '0';
            else if (y == 10)
                ch[cnt++] = 'A';
            else if (y == 11)
                ch[cnt++] = 'B';
            else if (y == 12)
                ch[cnt++] = 'C';
            else if (y == 13)
                ch[cnt++] = 'D';
            else if (y == 14)
                ch[cnt++] = 'E';
            else if (y == 15)
                ch[cnt++] = 'F';
        }
        if (n < 0) printf("-");
        for (int i = cnt - 1; i >= 0; i--) {
            printf("%c", ch[i]);
        }
        printf("\n");
    }
    return 0;
}

(T0004)Choose and divide

#include <cstdio> 
using namespace std; 
const double up = 1e8;
const int maxn = 2e4 + 10;
double p, q, r, s;
double a[maxn], b[maxn];
int cnta, cntb;
double ans;
int main() {
    while (scanf("%lf%lf%lf%lf", &p, &q, &r, &s) != EOF) {
        cnta = cntb = 0;
        ans = 1.0;
        for (int i = 1; i <= q; i++) a[i] = i + p - q;
        for (int i = q + 1; i <= q + s; i++) a[i] = i - q;
        for (int i = 1; i <= q; i++) b[i] = i;
        for (int i = q + 1; i <= q + s; i++) b[i] = i + r - s - q;
        int len = q + s;
        while (cnta != len || cntb != len)
            if (ans < up && cnta < len)
                ans *= a[++cnta];
            else
                ans /= b[++cntb];
        printf("%.5f\n", ans);
    }
    return 0;
}

如果直接暴力的计算，肯定会爆double精度的，那手写高精度吗？其实不用。 我们发现题目保证了最终的结果是小于1e8的，那我们可以考虑边乘边除的运算，只要乘的过程中 结果大于了1e8就开始除，否则就不停的乘。

(T0005)Letters Shop

Output:
For each friend print the length of the shortest prefix of letters from s s/he would need to buy to be able to construct her/his name of them. The name can be constructed if each letter is presented in the equal or greater amount.

It is guaranteed that every friend can construct her/his name using the letters from the string s.

Sample Input:

9 arrayhead 5 arya harry ray r areahydra

Sample Output:

 5 6 5 2 9

解：

#include <algorithm>
#include <cstdio>
#include <cstring>
using namespace std;
typedef long long ll;
const int maxn = 2e5 + 10;
int n, m;
char str[maxn];
int a[26][maxn], b[26], c[26];
int main() {
    scanf("%d", &n);
    scanf("%s", str);
    for (int i = 0; i < 26; i++) c[i] = 0;
    for (int i = 0; i < n; i++) {
        int x = str[i] - 'a';
        c[x]++;
        a[x][c[x]] = i + 1;
    }
    scanf("%d", &m);
    for (int i = 1; i <= m; i++) {
        int ans = 0;
        for (int i = 0; i < 26; i++) c[i] = 0;
        scanf("%s", str);

        int len = strlen(str);
        for (int j = 0; j < len; j++) {
            int x = str[j] - 'a';
            c[x]++;
            ans = max(ans, a[x][c[x]]);
        }
        printf("%d\n", ans);
    }
    return 0;
}

(T1014)F-Chessboard-CF961C*1400

题意
给出国际象棋棋盘（黑白相间）的四个大小为n×n（n为奇数）的部分，通过改变尽量少的格子的颜色，使得四个部分重新移动（不能旋转或翻转）并组合后能够得到一块完整的2n×2n大小的棋盘。
题解
因为n一定为奇数，所以组合前的四个部分一定是两个是四角都是黑色的、两个是四角都是白色的棋盘，而且这两种棋盘每一格的颜色正好相反，所以任意一块棋盘染成这两种需要的次数之和一定是n2接下来只需要枚举哪两块棋盘染成四角黑色、哪两块染成四角白色，找到这些方案的最小值即可。

#include <cstdio>

const int masks[6] = {3, 6, 12, 9, 5, 10};

int main() {
    int n, cnt[4] = {}, ans = 50000;
    scanf("%d", &n);
    for (int i = 0; i < 4; ++i) {
        char s[105];
        for (int j = 0; j < n; ++j) {
            scanf("%s", s);
            for (int k = 0; k < n; ++k) cnt[i] += (s[k] ^ j ^ k) & 1;
        }
    }
    for (int i = 0; i < 6; ++i) {
        int tmp = 0;
        for (int j = 0; j < 4; ++j)
            tmp += masks[i] & 1 << j ? cnt[j] : n * n - cnt[j];
        if (tmp < ans) ans = tmp;
    }
    printf("%d\n", ans);
    return 0;
}

（T1015)P1086 花生采摘（模拟，搜索）

鲁宾逊先生有一只宠物猴，名叫多多。这天，他们两个正沿着乡间小路散步，突然发现路边的告示牌上贴着一张小小的纸条：“欢迎免费品尝我种的花生！”。

int main(){
    cin>>m>>n>>t;
    for(i=1;i<=m;i++)
        for(j=1;j<=n;j++){
            cin>>a[i][j];
            if(a[i][j]>0){
                p[k].w=a[i][j];
                p[k].x=i;
                p[k].y=j;
                k++;
            }
        }
    for(i=1;i<k;i++)
        for(j=i+1;j<=k;j++)
            if(p[i].w<p[j].w)
                swap(p[i],p[j]);
    for(i=1;i<=k;i++){
        u=p[i].x;
        if(i==1) p[i].time=p[i].x+1;
        else p[i].time=p[i-1].time+abs(p[i].x-p[i-1].x)+abs(p[i].y-p[i-1].y)+1;
        if(p[i].time+u<=t) ans+=p[i].w;
    }
    cout<<ans;
    return 0;
}

(T1016)P1098（字符串的展开）

题目：

字符串展开有以下几种情况：

  • 在输入的字符串种出现了”-“号，减号两侧同为小写字母或同为数字，且按ASCII码的顺序，减号右边的字符严格大于左边的字符。
  • 参数p1：展开方式。p1=1时，对于字符子串，填充小写字母；p1=2时，对于字符字串，填充大写字母，这两种情况下数字字串的填充方式相同。p1=3时，不论是字母字串还是数字字串，都用与要填充的字母个数相同的星号”*”来填充。
  • 参数p2：填充字符的重复个数。p2=k表示同一个字符要连续填充k个。例题当p2=3时，子串”d-h”严格拓展为”deeefffgggh”。减号两边的字符不变。
  • 参数p3：是否改为逆序：p3=1表示维持原来的顺序，p3=2表示采用逆序输出，注意这时候仍然不包括减号两端的字符。例如当p1=1.p2=2,p3=2时，字串”d-h”应拓展为”dggffeeh”。
  • 如果减号右边的字符恰好是左边字符的后继，只删除中间的减号，例如：”d-e”应输出为”de”。如果减号右边的字符按照ASCII码的顺序小于或等于左边的字符，输出时，要保留中间的减号，例如：”d-d”应输出为”d-d”。

输入：

第一行为空格隔开的3个正整数，p1,p2,p3。

第二行为一行字符串，仅由数字，小写字母和减号”-“组成。行首和行末均无空格。

输出：

展开后的字符串。

样例：

输入：

1 2 1
abcs-w1234-9s-4zz

输出：

abcsttuuvvw1234556677889s-4zz

解：

#include <bits/stdc++.h>
using namespace std;
int p1,p2,p3,i=0,k;
char ch[300],be,af,f,j,p;
int main(){
    scanf("%d%d%d%s",&p1,&p2,&p3,ch);
    while(ch[i]){
        be=ch[i-1];
        af=ch[i+1];
        f=ch[i];
        if(f=='-'&&af>be&&(be>='0'&&af<='9')||(be>='a'&&af<='z')){
            for(p3==1?j=be+1:j=af-1;p3==1?j<af:j>be;p3==1?j++:j--){
                p=j;
                if(p1==2)
                    p=(p>='a')?p-32:p;
                else if(p1==3) p='*';
                for(k=0;k<p2;k++)
                    printf("%c",p);
            }
        }
        else
        {
            printf("%c",f);
        }
        i++;

    }
    return 0;
}

(T1017)P1149 火柴棒等式(搜索)

题目：

给你n根火柴棍，你可以拼出多少个形如”A+B=C”的等式？等式种的A,B,C是用火柴棍拼出的整数（若该数非零，则最高位不能是0）。用火柴棍拼数字0~9的拼法：

注意：

1、加号与等号各自需要两根火柴棍

2、如果$A\neq B$，则$A+B=C$与$B+A=C$视为不同的等式（$A,B,C \geq 0$）

3、n根火柴棍必须全部用上

输入：

一个整数n($n\leq 24$)。

输出：

能拼成的不同等式的数目。

输入：

14

输出：

2

解：

#include<bits/stdc++.h>
using namespace std;
int cnt[10]={6,2,5,5,4,5,6,3,7,6};
int n;
int main(){
    cin>>n;
    int j;
    int a[2001];
    int _cnt=0;
    a[0]=6;
    for(int i=1;i<=2000;i++){
        j=i;
        while(j>=1){
            a[i]=a[i]+cnt[j%10];
            j/=10;
        }
    }
    for(int i=0;i<=1000;i++)
        for(int j=0;j<=1000;j++)
            if(a[i]+a[j]+a[i+j]+4==n) _cnt++;
    cout<<_cnt<<endl;
    return 0;
}

(T1018)P1036（选数）

题目：

已知n个整数$x_1,x_2,…x_n$，以及1个整数k(k<n)。从n个整数种任选k个整数相加，可分别得到一系列的和。现在要求你计算出和为素数共有多少种。

输入：

第一行：$n,k,(1\leq n \leq 20,k<n)$

第二行：$x_1,x_2,…x_n$$(1\leq x_i\leq 5000000)$

输出：

满足条件的整数。

输入：

4 3
3 7 12 19

输出：

1

解：

搜索种组合数的问题

#include <bits/stdc++.h>
using namespace std;
int f[21];
int n,k;
int vis[21];
int _cnt=0;
int sum=0;
int d[21];
bool is_prime(int num){
    int s=sqrt(double(num));
    for(int i=2;i<=s;i++){
        if(num%i==0) return false;
    }
    return true;
}
void dfs(int i,int st){
    if(i==k){
        if(is_prime(sum)) _cnt++;
        return;
    }
    for(int j=st;j<n;j++){
            d[i]=f[j];
            sum+=f[j];
            dfs(i+1,j+1);
            sum-=f[j];

    }
}
int main(){
    sum=0;
    memset(vis,0,sizeof(vis));
    cin>>n>>k;
    for(int i=0;i<n;i++){
        cin>>f[i];
    }
    _cnt=0;
    dfs(0,0);
    cout<<_cnt<<endl;
    return 0;
}
