38.外观数列

题目链接 https://leetcode-cn.com/problems/count-and-say/

比较经典的一道题目了，南邮以前有个比赛好像出到过这样类似的题。

大概意思就是根据前一个数字串来生成一个新的数字串。

例如:

21: 一个2，一个1，故生成为 1211

class Solution {
public:
    string countAndSay(int n) {
        string st="1";
        for(int i=0;i<n-1;i++){
            string ans;
            for(int j=0;j<st.size();j++){
                int k=j;
                while(st[k]==st[j]&&k!=st.size()) k++;
                ans+=to_string(k-j);
                ans+=st[j];
                j=k-1;
            }
            st=ans;
        }
        return st;
    }
};

先说一下LeetCode是怎么用的，这个跟我们以往的OJ不同，它不需要你手动输入数据，然后从头到尾把程序完整地写下来。它只提供给你一个类和一个函数，输入文件通过参数传入，你的结果通过return语句返回。结合这个程序应该很好理解。

然后说一下这题的思路，主要难点就在于对重复字符的判断，这里我们通过一个循环，用j和k分别表示重复字符串的头尾，就好处理了。

49.字母异位词分组

题目链接https://leetcode-cn.com/problems/group-anagrams/

典型的哈希类型的题目，将字母排序，然后将排序结果一致的放入一个哈希表里面，输出哈希表就行了。

class Solution {
public:
    vector<vector<string>> groupAnagrams(vector<string>& strs) {
        unordered_map<string,vector<string>> hash;
        for(auto s : strs){
            string st=s;
            sort(st.begin(),st.end());
            hash[st].push_back(s);
        }
        vector<vector<string>> vec;
        for(auto item : hash)  vec.push_back(item.second);
        return vec;
    }
};

151.翻转字符串里的单词

题目链接 https://leetcode-cn.com/problems/reverse-words-in-a-string/

这个题目的话，如果有知道一个叫做 stringstream的类就比较好做，难点在于消除单词间的空格。

当然，也可以用类似第一题的方法消除连续的空格。

class Solution {
public:
    string reverseWords(string s) {
        stringstream ss(s);
        vector<string> vec;
        string str;
        while(ss>>str) vec.push_back(str);
        string ans;
        for(int i=vec.size()-1;i>=0;i--){
            if(i==0) ans+=vec[i];
            else ans+=vec[i]+' ';
        }
        return ans;
    }
};
