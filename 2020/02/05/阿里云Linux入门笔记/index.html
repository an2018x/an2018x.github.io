阿里云Linux入门笔记

Ubuntu 安装

建议手动分区

主要目录：

  • /:系统根目录，建议容量为磁盘的5%，最大20GB
  • /usr:包含一般不需要修改的应用程序，命令程序文件，程序库，手册，建议容量为磁盘的5%，最大20GB
  • /var:存放系统产生的经常变换的文件，建议容量为硬盘的3%，最大10GB
  • /home:存放用户文件的主目录，用户数据，建议容量为硬盘的3%，最大10GB
  • /tmp:存放启动时产生的临时文件，建议容量为磁盘的1%，最大5GB
  • swap:交换区。建议容量为内存2倍及以上

进入系统后：

第一个命令：sudo su，登陆高级管理员（sudo，全称superuser do，即超级管理员）

关机：init 0

参数 意义
0 停机
1 单用户模式，只root进行维护
2 多用户，不能使用net file system
3 完全多用户
4 安全模式
5 图形化
6 重启

使用最多：0和6

如果不是在root用户下，则是sudo init 0

基本命令

组成部分：

  • 命令主体(command)
  • 命令选项[option]，说明对命令的要求
  • 命令其他参数[argument]，描述命令作用的对象

[]表示命令可有可无

command [option] [argument]

注意事项：

  • 命令区分大小写

  • 使用tab键补全命令剩余部分

  • 使用帮助手册：find - -help

  • 命令基本格式：

    命令 [参数 参数] … find /etc -name *pass*

  • 程序文件或shell脚本能否执行与文件拓展名无关，与文件基本属性有关

ls --help 查看ls的帮助
ls --help |more  管道，把前一个命令处理的结果传递到后一个命令，后一个命令再处理，翻页
ctrl + C 直接强制退出程序
ctrl + D 退出当前用户

命令提示符表示用户输入的状态：

  • 系统管理员用户 root: linux-001:/#
  • 系统普通用户：web@linux-001:/>

命令的每个组成部分之间必须用空格隔开

命令的输入以回车键结束

用vi编辑器打开interfaces文件

address 192.168.68.111    静态ip地址
netmask  255.255.255.0     子网掩码
gateway   192.168.68.2      网关

etc/init.d/networking restart

配置dns

cd etc/resolvconf/resolve.conf.d
vi base
nameserver 192.168.68.2   用路由器网关作为第一个
