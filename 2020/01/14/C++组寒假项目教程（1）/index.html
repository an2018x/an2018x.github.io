编写菜单栏

什么是菜单栏？文件、编辑等在的区域就是菜单栏，也叫menubar。
看这部分前，建议先看看我发的书的第86页，大概是5.1的内容。

接下来的代码就是如何用代码添加菜单栏，结合书籍自己看看就能领会了，这里不多讲

    myMenuBar=new QMenuBar(this);
    //增加菜单栏
    file=myMenuBar->addMenu(tr("文件(&F)"));//这里&F的作用，自己查找
    edit=myMenuBar->addMenu(tr("编辑(&E)"));
    select=myMenuBar->addMenu(tr("选择(&S)"));
    view=myMenuBar->addMenu(tr("查看(&V)"));
    go=myMenuBar->addMenu(tr("转到(&G)"));
    debug=myMenuBar->addMenu(tr("调试(&D)"));
    terminal=myMenuBar->addMenu(tr("终端(&T)"));
    help=myMenuBar->addMenu(tr("帮助(&H)"));
    //细分菜单栏
    QAction *actionNew=file->addAction(tr("新建文件"));
    QAction *actionNewWindow=file->addAction(tr("新建窗口"));
    actionNew->setShortcut(QKeySequence("Ctrl+N"));
    actionNewWindow->setShortcut(QKeySequence("Ctrl+Shift+N"));
    file->addSeparator();
    QAction *actionOpen=file->addAction(tr("打开文件"));
    QAction *actionOpenDir=file->addAction(tr("打开文件夹"));
    actionOpen->setShortcut(QKeySequence("Ctrl+O")); //快捷键
    actionOpenDir->setShortcut(QKeySequence("Ctrl+Shift+O"));
    file->addSeparator();

qss模仿了一部分css语法，但功能没有它强大，主要用于界面的设计。通过setstylesheet()函数来应用样式。

    qssFile = new QFile(":/stylesheet.qss", this);
    // 只读方式打开该文件
    qssFile->open(QFile::ReadOnly);
    // 读取文件全部内容，使用tr()函数将其转换为QString类型
    QString styleSheet = tr(qssFile->readAll());
    // 为QApplication设置样式表
    qApp->setStyleSheet(styleSheet);
    qssFile->close();

重写QPlainTextEdit

QPlainTextEdit是qt自带的多行文本编辑器，由于我们这个要支持高亮等功能，所以我们需要重写这个类，也就是用一个新类来继承它。不了解的先看看C++继承方面的知识。

#ifndef CODEEDIT_H
#define CODEEDIT_H
#include <QPlainTextEdit>


class CodeEdit : public QPlainTextEdit
{
    Q_OBJECT
public:
    CodeEdit(QWidget *parent = 0);
};

#endif // CODEEDIT_H
#include "codeedit.h"


CodeEdit::CodeEdit(QWidget *parent): QPlainTextEdit(parent)
{
    this->setFont(QFont(tr("Consolas"), 14));
    this->setStyleSheet("background-color:rgb(33,33,33);color:white;");
}

实例化我们的新类：

    codeArea=new CodeEdit(this);
    codeArea->resize(1000,1000);
    codeArea->move(60,35);
