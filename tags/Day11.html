P1090合并果子

在一个果园里，多多将所有的果子打了下来，而且按果子的不同种类分成了不同的堆。多多决定把所有的果子合并到一堆。

每一次合并，多多可以把两堆果子合到一起，消耗的体力等于两堆果子的重量之和。可以看出，所有的果子在经过$n-1$次合并后，就只剩下一堆了。多多在合并果子时总共消耗的体力等于每次合并所消耗的体力之和。

因为还要花大力气把这些果子搬回家，所以多多在和并果子时要尽可能地节省体力。假定每个果子的重量都为1，并且已知果子的种类数和每种果子的数目，你的任务是设计出合并的次序方案，式多多耗费的体力最少，并输出这个最少的体力耗费值。

输入格式

第一行：整数n$(1\leq n \leq 10000)$表示果子的个数

第二行：包含n个整数，用空格分隔，第$i$个整数$a_i(1\leq a_i \leq 20000)$是第$i$种果子的数目

输出格式

最小的体力耗费值
